This program needs:
    python 3.8
    pygame 2
    random.randint()
    time.sleep()
    sys.exit()

How to play:
    In menu move with arrow keys and select with a space.
    In game move with A, S and jump with W.
    When you find the doors press space.

All images was painted in pikelapp(https://www.piskelapp.com/)
All sounds was created using sfxr
All colors was taken from lospec(https://lospec.com/palette-list)
